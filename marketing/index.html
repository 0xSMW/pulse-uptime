9.0   Harden

            
Trust nothing

          

          

            Every target, response, redirect, and DNS answer is treated as
            hostile by design.
          

        


        

          

            

              
SSRF-proof checking

              

                Only publicly routable addresses. Every DNS answer is validated
                and the socket is pinned to it.
              

            

            

              
Hardened auth

              

                Argon2id passwords, gated first-admin bootstrap, rate-limited
                login, hashed tokens with lineage.
              

            

            

              
Hostile-proof CLI

              

                pulsectl
                treats server responses as untrusted and rejects control
                characters outright.

              enforcing

Untrusted providers
